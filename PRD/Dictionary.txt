1) APP NAME / GLOBAL
============================================================
App name (display): მოსავალი
Currency: ₾ (GEL)
Weight unit input: კგ
Weight unit display (secondary): ტ (ტონა)

Number formatting:
- Use comma for decimals if you follow Georgian locale (optional). If not supported, use dot consistently.
- Always display currency with ₾ (e.g., 120 ₾)
- Display kg as “კგ” and tons as “ტ”
